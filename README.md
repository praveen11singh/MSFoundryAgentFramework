# MSFoundryMSAFramework

This project provides examples and implementations for the Microsoft Agent Framework, demonstrating various agent configurations and interactions.

## Project Structure

- `msagentframework_multiagent.py`: Example implementation of a multi-agent system using the Microsoft Agent Framework.
- `msagentframework_multiturn_conversation.py`: Demonstration of multi-turn conversations with agents.
- `msagentframework_singleagent.py`: Basic example of a single agent setup.

## Prerequisites

- Python 3.8 or higher
- Microsoft Agent Framework SDK

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd MSFoundryMSAFramework
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the example scripts directly with Python:

```
python msagentframework_singleagent.py
```
